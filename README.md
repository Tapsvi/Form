# Contact Us Form
<h3><b>About</b></h3>
 This project accepts entered contact details and validates it with the mentioned validation specifications and submits your contact details to the concerned clients of the application.
<h3><b>Use Case</b></h3>
 Wherever you need to make use of forms in your applications to fetch contact details of users, this could be useful.This could also be useful for the beginners who has keen interest to learn Javascript Form validation concepts and DOM manipulations.
